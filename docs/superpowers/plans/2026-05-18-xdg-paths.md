# XDG Paths Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Move `repos.toml` to `$XDG_DATA_HOME/liszt/` and clone cache to `$XDG_CACHE_HOME/liszt/`; keep `liszt.toml` and `liszt.lock` at the cwd.

**Architecture:** New `internal/xdg` package resolves user dirs with XDG env-var precedence and home-dir fallback. `cmd/liszt/main.go` builds the `cli.Paths` struct from `xdg.DataDir()` and `xdg.CacheDir()`. `repos.Save`, `lock.Save`, and `manifest.Save` gain a `MkdirAll(parent)` call so writes succeed when the parent directory does not exist yet. No tests (per user decision); verification is `go build` + a smoke run.

**Tech Stack:** Go 1.26, `github.com/pelletier/go-toml/v2`, stdlib `os` / `path/filepath`.

**Spec:** `docs/superpowers/specs/2026-05-18-xdg-paths-design.md`

- [ ] **Step 1: Write `internal/xdg/xdg.go`**

```go
package xdg

import (
	"os"
	"path/filepath"
)

// DataDir returns the liszt data directory.
// $XDG_DATA_HOME/liszt if XDG_DATA_HOME is set to an absolute path,
// otherwise ~/.local/share/liszt.
func DataDir() string {
	return resolve("XDG_DATA_HOME", filepath.Join(".local", "share"))
}

// CacheDir returns the liszt cache directory.
// $XDG_CACHE_HOME/liszt if XDG_CACHE_HOME is set to an absolute path,
// otherwise ~/.cache/liszt.
func CacheDir() string {
	return resolve("XDG_CACHE_HOME", ".cache")
}

func resolve(envVar, homeRel string) string {
	if v := os.Getenv(envVar); v != "" && filepath.IsAbs(v) {
		return filepath.Join(v, "liszt")
	}
	home, err := os.UserHomeDir()
	if err != nil {
		home = "."
	}
	return filepath.Join(home, homeRel, "liszt")
}
```

```go
// Save writes cfg to path. Creates parent directories as needed.
func Save(path string, cfg *Config) error {
	data, err := toml.Marshal(cfg)
	if err != nil {
		return err
	}
	if err := os.MkdirAll(filepath.Dir(path), 0755); err != nil {
		return err
	}
	return os.WriteFile(path, data, 0644)
}
```
